---

Morgen is a free morning dashboard app that helps you start every day with focus. It shows your important links, weather, world clocks, website status, and subscriptions in one clean page. Everything runs on your computer. No account needed, no data leaves your machine.

Designed and built by <a href="https://conceptuel.ch" target="_blank">Robbie Conceptuel</a>.
